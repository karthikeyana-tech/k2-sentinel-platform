# K² Sentinel Domain Model

## Core Entities

### Tourist
Represents a registered tourist using the platform.

### Trip
Represents a travel itinerary.

### Emergency Contact
Stores emergency contact information.

### GeoFence
Represents a monitored geographical zone.

### Risk Assessment
Represents the calculated safety score.

### Incident
Represents an emergency or safety event.

### Alert
Represents notifications generated by the system.

### Authority
Represents government officials using the platform.
